# 仓库地图系统 - 快速入门

## 5分钟快速开始

### 步骤1：访问地图页面

在浏览器中访问系统，导航到 **仓储管理 → 仓库地图**，或直接访问 `/wms/map`

### 步骤2：创建一个简单地图

1. 在左侧"地图设置"中，保持默认设置：
   - 地图宽度：15列
   - 地图高度：10行
   - 格子大小：50px
   - 格子间隙：5px

2. 点击"重新生成地图"按钮

你将看到一个地图，第一行有一个绿色的"大门"格子。

### 步骤3：设置库区

1. 在"库区管理"区域，默认已有几个库区（A区、B区、C区、D区）

2. 在地图上，**右键点击**任意格子（除了大门）

3. 在弹出菜单中选择一个库区，例如"A区"

4. 重复此操作，设置更多格子为不同的库区

**建议设置示例**：
- 左上角区域：设置为"A区"（蓝色）
- 右上角区域：设置为"B区"（橙色）
- 左下角区域：设置为"C区"（红色）
- 右下角区域：设置为"D区"（灰色）

### 步骤4：规划路径

1. 在"路径规划"区域，点击"选择目标库区"下拉框

2. 选择你想要访问的库区，例如：A区、B区、C区

3. 点击"计算最优路径"按钮

4. 等待1-2秒，系统会显示：
   - 地图上的最优路径（蓝色虚线）
   - 访问顺序（0、1、2...）
   - 总距离

### 步骤5：播放动画

1. 在"路径动画"区域，调整动画速度（建议选5）

2. 点击"播放动画"按钮

3. 观看黄色标记沿着路径移动，访问每个库区

4. 动画结束后，可以点击"重置"再次播放

## 常见使用场景

### 场景1：优化拣货路线

**问题**：仓库有10个订单，需要从A、B、C、D、E五个库区拣货，如何规划最短路线？

**解决方案**：
1. 在地图上标记出A、B、C、D、E五个库区的位置
2. 在"路径规划"中选择这五个库区
3. 点击"计算最优路径"
4. 系统会给出最优访问顺序和路线

### 场景2：新员工培训

**问题**：新员工不熟悉仓库布局，需要快速了解各库区位置和拣货路线

**解决方案**：
1. 打开已配置好的仓库地图
2. 使用"鼠标悬停"功能，让员工熟悉各库区位置
3. 演示常用拣货路线的动画
4. 让员工自己尝试规划路径

### 场景3：仓库布局优化

**问题**：需要重新规划仓库布局，减少拣货距离

**解决方案**：
1. 创建当前布局的地图
2. 记录常用拣货路线的总距离
3. 尝试调整库区位置（重新右键设置）
4. 再次计算路径，比较距离
5. 选择最优布局方案

## 实用技巧

### 技巧1：快速设置连续格子

想要将多个相邻格子设置为同一库区：
1. 右键第一个格子，选择库区
2. 右键第二个格子，选择同一库区
3. 重复此操作（会越来越快）

### 技巧2：查看格子信息

不确定某个格子属于哪个库区？
- 只需将鼠标移动到格子上，会显示浮窗信息

### 技巧3：清除错误设置

设置错了库区？
- 右键点击格子，在菜单中选择"清除"

### 技巧4：调整算法参数

路径不够精确？
- **小地图**：减少迭代次数（5000），加快计算
- **大地图**：增加迭代次数（20000），提高精度
- **复杂路线**：提高初始温度（2000+），增加冷却速率（0.997+）

### 技巧5：对比不同方案

想对比多种路线方案？
1. 计算第一个方案，记录距离
2. 点击"清除路径"
3. 重新选择目标库区，计算新方案
4. 比较距离，选择最优方案

## 快速参考

### 鼠标操作
- **左键点击**：选择/查看
- **右键点击**：打开设置菜单
- **鼠标悬停**：显示信息浮窗

### 颜色含义
- **绿色**：大门/起点
- **蓝色**：库区/中间点
- **红色**：终点
- **黄色**：动画当前位置
- **灰色**：未设置的空格子

### 按钮功能速查

| 按钮 | 功能 |
|-----|-----|
| 重新生成地图 | 根据新的行列数重建地图 |
| 清空设置 | 清除所有格子的库区设置 |
| 添加库区 | 创建新的库区 |
| 计算最优路径 | 运行模拟退火算法 |
| 清除路径 | 移除当前显示的路径 |
| 播放动画 | 动态展示路径 |
| 停止 | 暂停动画播放 |
| 重置 | 重置动画到开始 |
